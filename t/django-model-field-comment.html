django model字段添加备注Comment

mysql中create创建表时可以通过COMMENT给字段添加备注，例如下边这样：

> show create table ops_coffee;
| ops_coffee | CREATE TABLE `ops_coffee` (
  `id` varchar(20) NOT NULL COMMENT 'id',
  `name` varchar(255) NOT NULL COMMENT '名字',
  PRIMARY KEY (`id`)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4

django model本身没有这个功能，如果你非要这么做的话可以这样:

  1. 执行makemigrations命令生成文件，然后进去修改创建表的语句

  2. 再执行migrate创建表
